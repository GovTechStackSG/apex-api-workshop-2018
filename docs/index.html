attack

Attack other teams

attack

Use a specified weapon to attack another team, and bring down their health

POST /attack
AttackRequest

Specifies the attacker, defender, weapon used, and provides the attacker password

Request Content-Types: application/json
Request Example
{
  "attacker": "team00",
  "defender": "team99",
  "password": "only-the-attacker-can-know-this",
  "weaponName": "snowball"
}
200 OK

JSON object including attacking and defending team's updated status, weapon used, and how many damage was done during API call

400 Bad Request

Attacking/defending team has no more health; or attacking team has not enough ammo; or attacking team provided an invalid password
