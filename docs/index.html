Attack

Up 
post /attack
Use a specified weapon to attack another team, and bring down their health (attackPost)

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

attack AttackRequest (required)
Body Parameter — Specifies the attacker, defender, weapon used, and provides the attacker password

Responses

200

JSON object including attacking and defending team's updated status, weapon used, and how many damage was done during API call

400

Attacking/defending team has no more health; or attacking team has not enough ammo; or attacking team provided an invalid password

Blacksmith

Up 
get /blacksmith/levels/1
Obtain a programming puzzle. The answer should be submitted to this endpoint via POST, in order to obtain an Apex L1 secret (blacksmithLevels1Get)

Return type

BlacksmithL1Question

Example data

Content-Type: application/json
{
  "string" : "hello",
  "text" : "Reverse the given string"
}

Responses

200

The puzzle to solve to obtain L1 secrets BlacksmithL1Question

404

Invalid authentication level
Up 
post /blacksmith/levels/1
Submit the answer to the L1 programming puzzle (blacksmithLevels1Post)

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

answer BlacksmithL1Answer (required)
Body Parameter — Answer to the puzzle issued by the GET API.

Return type

L1Secret

Example data

Content-Type: application/json
{
  "appId" : "my-apex-app-id",
  "secret" : "my-apex-app-secret"
}

Responses

200

Answer verified L1Secret

400

No answer or invalid answer submitted

404

Invalid authentication level
Up 
get /blacksmith/levels/2
Obtain a programming puzzle that requires the answer to the L1 puzzle. The answer should be submitted to this endpoint via POST, in order to obtain an Apex L2 secret (blacksmithLevels2Get)

Return type

BlacksmithL2Question

Example data

Content-Type: application/json
{
  "text" : "Return a capitalized version of the L1 answer"
}

Responses

200

The puzzle to solve to obtain L2 secrets BlacksmithL2Question

404

Invalid authentication level
Up 
post /blacksmith/levels/2
Submit the answer to the L2 programming puzzle (blacksmithLevels2Post)

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Request body

answer BlacksmithL2Answer (required)
Body Parameter — Answer to the puzzle issued by the GET API.

Return type

L2Secret

Example data

{

}

200

Teams

Up 
get /teams/status
Get statuses for all teams (teamsStatusGet)

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json

Return type

array[Team]

Example data

Content-Type: application/json
[ {
  "name" : "team99",
  "health" : 1000,
  "ammoL1" : 5,
  "id" : 0,
  "ammoL0" : 5,
  "ammoL2" : 5
}, {
  "name" : "team99",
  "health" : 1000,
  "ammoL1" : 5,
  "id" : 0,
  "ammoL0" : 5,
  "ammoL2" : 5
} ]

Responses

200

Array of statuses for all teams
Up 
get /teams/{teamName}/status
Get status for a single team (teamsTeamNameStatusGet)

Path parameters

teamName (required)
Path Parameter — Name of team to get status of

Return type

Team

Example data

Content-Type: application/json
{
  "name" : "team99",
  "health" : 1000,
  "ammoL1" : 5,
  "id" : 0,
  "ammoL0" : 5,
  "ammoL2" : 5
}

Responses

200

JSON object containing information about a particular team Team
